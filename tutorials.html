OiiOii 创作指南（内测 V1.1）总览

创作前：准备与配置

  • 明确目标：场景（自媒体/教育/企业/娱乐）、风格（二次元/Lo‑Fi/黏土/3D）、时长与分发渠道
  • 账户与权限：申请内测并完成设置；了解生成次数与分辨率上限
  • 素材与规范：准备品牌元素（Logo/色彩/字体）、角色三视图与关键词词库

创作中：七大 Agent 流程

艺术总监 → 编剧 → 分镜师 → 角色设计师 → 场景设计师 → 动画师 → 音效师。各环节职责明确，统一风格与节奏。

  • 艺术总监：设定风格范式（色彩/线条/颗粒感/节奏）
  • 编剧：将需求转化为可分镜脚本（场景、动作、台词、镜头节奏）
  • 分镜师：生成故事板（镜头类型、时长、转场、关键元素），支持 PDF 导出
  • 角色/场景：输出角色三视图与基础动作、搭建环境细节与氛围元素
  • 动画师：合成角色与场景，控制镜头运动与转场，输出 1080p 成片
  • 音效师：匹配配乐与音效，支持多语言配音与音画同步

提示词策略与模板

采用“场景 + 风格 + 时长 + 角色 + 动作 + 镜头 + 音效”结构；结合模板库与智能生成器，快速生成高质量提示词。

  • 基础模板：短视频、美食探店、儿童科普、企业宣传
  • 进阶模板：系列 IP 复用、风格迁移、跨平台投放变体
  • 语义增强：加入情绪、节奏与镜头动线词汇，提升生成稳定度

场景实操路径

按场景选择模块并执行标准流程：

  • 自媒体：批量生产、定时发布、系列角色一致性
  • 教育：知识点标注、字幕与配音、课堂节奏优化
  • 企业：品牌统一、卖点拆解、培训与产品演示
  • 娱乐/衍生：生日祝福、情侣纪念、IP 衍生品设计与预览

质量与效率

  • 质量控制：风格与角色一致性校验、音画同步与字幕规范
  • 效率提升：模板化+资产复用，支持“1 人 1 天 20 集动画”产能
  • 导出与分发：1080p 默认输出，多语言字幕与平台适配

常见问题与故障排查

  • 风格跑偏：检查艺术总监参数与提示词风格词汇一致性
  • 镜头节奏不稳：优化分镜时长与转场类型，减少无效镜头
  • 角色不一致：复用角色库与三视图提示词，禁用模糊表述
  • 音画不同步：规范拍点与动作节点，统一帧率与节拍

视频创作

快速生视频

  • 选择首页【快速生视频】工具，输入灵感即可生成
  • 参考图片：最多 7 张，可作为风格/角色参考
  • 修改方式：在对话输入修改意见或点击画布的重新生成

音乐概念短片

  • 选择【音乐概念短片】；依次选择类型、横竖屏、时长、语言、风格、情绪
  • 自动生成分镜与成片，支持 7 张参考图
  • 修改：与 Agent 对话指定角色/分镜，或在画布编辑/重新生成/剪辑

剧情故事短片

  • 选择【剧情故事短片】；设置横竖屏、时长、语言、风格、情绪
  • 导演风格：在提示词加入导演名并上传风格参考图（可选）
  • 科普视频：明确步骤标注，系统自动生成脚本、分镜与成片
  • 系列动画：使用资产库角色，保证角色一致性

漫画转视频

  • 上传分镜清晰、对话简短的漫画图片
  • 输入灵感并等待生成动画片段

图片创作

  • 在对话框输入生图要求即可生成
  • 支持角色参考图与风格参考图
  • 支持模型：Midjourney、GPT‑4o、Nano Banana、Novel、Seedance 4.0
  • 推荐玩法：生成周边谷子图片（上传参考图并描述类型）

角色创作

  • 在对话框输入角色创作要求，生成角色与三视图
  • 加入资产库后可长期用于视频/图片创作

风格库与作品示例

  • 风格参考：Minecraft、Ghibli、Lo‑Fi Chibi、Clay Toy、Vaporwave 等
  • 作品类型：音乐短片、故事短片、系列动画、MV、快速生图等
