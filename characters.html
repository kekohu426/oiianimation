角色提示词 FAQ

如何用角色 Prompt 提升 AI 生成质量？

结合常见提问总结的 Prompt 使用策略，同时提供内部链接帮助搜索引擎理解页面主题。

Prompt / Negative 怎么搭配？

默认展示的 Prompt 与 Negative 已经按角色设定调优，建议先完整复制，再根据场景替换部分服装或表情关键词，保留 Negative 避免多余元素。

适配哪些模型？

提示词可用于 CogView、Flux、SDXL 等主流模型。想要更写实的表现可尝试 Flux Realism，国漫或厚涂风格可在 教程中心 查看推荐参数。

如何做 SEO / 二次创作？

复制 Prompt 后，在描述中加入行业或学科关键词即可快速出图，再配合 博客 中的 SEO 指南，提高角色内容的搜索权重。
